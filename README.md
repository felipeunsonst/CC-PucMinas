# CC - PUC Minas

Este repositório é destinado a armazenar meus trabalhos, provas, anotações e demais registros produzidos ao longo da minha graduação em Ciência da Computação.

É importante ressaltar que, no momento da criação deste repositório, eu já não possuía todos os materiais de algumas disciplinas cursadas. Assim, apenas os conteúdos que consegui recuperar estão disponíveis aqui.

Em cada pasta referente a um período, há um arquivo README informando todas as disciplinas cursadas naquele semestre, de forma que seja possível identificar quais matérias não possuem registros.
